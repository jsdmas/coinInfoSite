# description
주된 기능으로는 실시간으로 코인들의 가격정보를받아 차트로 표시해주는 사이트입니다.  
typescript를 익히기위해 처음으로 만들어본 사이트입니다.   
api 설정과 react-query의 기본적인 개념에대해 배우게 되었습니다.  
전체적인 State를 recoil을 통해 다루고 dark모드를 만들어봤습니다.    
react의 router설정에 대해 잘 알게 되었습니다.
